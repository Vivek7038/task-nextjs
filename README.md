# Nextjs Task

The application is responsive on both desktop and mobile.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Vivek7038/task-nextjs.git
```

Go to the project directory

```bash
  cd task-nextjs
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm run dev
```

Server will be live at
http://localhost:3000/
